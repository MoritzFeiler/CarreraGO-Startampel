# CarreraGO-Startampel
Schaltplan, Arduino Code und stl-Dateien zum Bau einer Miniatur-Startampel für Carrerabahnen und ähnliches.
## Material:
- Arduino Nano
- 10x RGB-LED (common anode)
- Lochrasterplatine 
- 5x 33 Ohm Widerstände
- 5x 180 Ohm Widerstände
- Piezo Buzzer
